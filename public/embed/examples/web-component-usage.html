This page demonstrates how to use the chatbot as a web component with data attributes for easy configuration.

Method 1: Data Attributes (Auto-initialization)

Simply add the script tag with data attributes and the chatbot will initialize automatically:

<!-- Auto-initialization with data attributes --> <script src="https://yourdomain.com/embed/chatbot-embed.js" data-chatbot-widget data-iframe-url="https://yourdomain.com/iframe" data-position="bottom-right" data-theme="dark" data-auto-open="false" data-toggle-button-text="Chat with us" data-toggle-button-icon="ðŸ’¬"> </script>

Method 2: Custom Web Component

Create a custom web component for more control:

<!-- Custom web component --> <chatbot-widget iframe-url="https://yourdomain.com/iframe" position="bottom-right" theme="dark" auto-open="false" toggle-button-text="Support" toggle-button-icon="ðŸ¤–"> </chatbot-widget> <script> // Define custom web component class ChatbotWidget extends HTMLElement { connectedCallback() { const iframeUrl = this.getAttribute('iframe-url'); const position = this.getAttribute('position') || 'bottom-right'; const theme = this.getAttribute('theme') || 'dark'; const autoOpen = this.getAttribute('auto-open') === 'true'; const toggleButtonText = this.getAttribute('toggle-button-text') || 'Chat'; const toggleButtonIcon = this.getAttribute('toggle-button-icon') || 'ðŸ’¬'; // Initialize chatbot with attributes if (window.ChatbotWidget) { window.ChatbotWidget.init({ iframeUrl, position, theme, autoOpen, toggleButtonText, toggleButtonIcon }); } } } customElements.define('chatbot-widget', ChatbotWidget); </script>

Method 3: React Component

For React applications, create a wrapper component:

import React, { useEffect, useRef } from 'react'; const ChatbotWidget = ({ iframeUrl, position = 'bottom-right', theme = 'dark', autoOpen = false, toggleButtonText = 'Chat with us', toggleButtonIcon = 'ðŸ’¬' }) => { const scriptRef = useRef(null); useEffect(() => { // Load the chatbot script const script = document.createElement('script'); script.src = 'https://yourdomain.com/embed/chatbot-embed.js'; script.async = true; script.onload = () => { // Initialize chatbot when script loads if (window.ChatbotWidget) { window.ChatbotWidget.init({ iframeUrl, position, theme, autoOpen, toggleButtonText, toggleButtonIcon }); } }; document.head.appendChild(script); scriptRef.current = script; return () => { // Cleanup if (scriptRef.current) { document.head.removeChild(scriptRef.current); } if (window.ChatbotWidget) { window.ChatbotWidget.destroy(); } }; }, [iframeUrl, position, theme, autoOpen, toggleButtonText, toggleButtonIcon]); return null; // The widget renders itself }; export default ChatbotWidget;

Method 4: Vue Component

For Vue applications:

<template> <div ref="chatbotContainer"></div> </template> <script> export default { name: 'ChatbotWidget', props: { iframeUrl: { type: String, required: true }, position: { type: String, default: 'bottom-right' }, theme: { type: String, default: 'dark' }, autoOpen: { type: Boolean, default: false }, toggleButtonText: { type: String, default: 'Chat with us' }, toggleButtonIcon: { type: String, default: 'ðŸ’¬' } }, mounted() { this.loadChatbot(); }, beforeUnmount() { this.destroyChatbot(); }, methods: { loadChatbot() { const script = document.createElement('script'); script.src = 'https://yourdomain.com/embed/chatbot-embed.js'; script.async = true; script.onload = () => { if (window.ChatbotWidget) { window.ChatbotWidget.init({ iframeUrl: this.iframeUrl, position: this.position, theme: this.theme, autoOpen: this.autoOpen, toggleButtonText: this.toggleButtonText, toggleButtonIcon: this.toggleButtonIcon }); } }; document.head.appendChild(script); }, destroyChatbot() { if (window.ChatbotWidget) { window.ChatbotWidget.destroy(); } } } }; </script>

Supported Data Attributes

  • data-iframe-url: URL to the chatbot iframe page
  • data-position: Widget position (bottom-right, bottom-left, top-right, top-left)
  • data-theme: Color theme (dark, light)
  • data-auto-open: Whether to open automatically (true, false)
  • data-toggle-button-text: Text for the toggle button
  • data-toggle-button-icon: Icon/emoji for the toggle button
  • data-show-toggle-button: Whether to show toggle button (true, false)
