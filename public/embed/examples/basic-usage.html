This page demonstrates the basic integration of the chatbot widget. The chatbot will appear as a floating button in the bottom-right corner.

Integration Steps:

  1. Include the chatbot embed script in your HTML
  2. Initialize the widget with basic configuration
  3. Customize the appearance and behavior as needed

Code Example:

<!-- Include the chatbot embed script --> <script src="https://yourdomain.com/embed/chatbot-embed.js"></script> <!-- Initialize the chatbot widget --> <script> // Basic initialization ChatbotWidget.init({ iframeUrl: 'https://yourdomain.com/iframe', position: 'bottom-right', theme: 'dark', autoOpen: false }); </script>

Configuration Options:

  • iframeUrl: URL to your chatbot iframe page
  • position: Widget position (bottom-right, bottom-left, top-right, top-left)
  • theme: Color theme (dark, light)
  • autoOpen: Whether to open chatbot automatically on page load
  • toggleButtonText: Text for the toggle button
  • toggleButtonIcon: Icon for the toggle button

The chatbot is now active on this page! Look for the floating button in the bottom-right corner.
