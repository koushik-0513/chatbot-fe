Event Handling

The chatbot widget supports various events and can be controlled programmatically:

// Event listeners window.addEventListener('message', function(event) { if (event.data.type === 'CHATBOT_READY') { console.log('Chatbot is ready'); } }); // Programmatic control ChatbotWidget.open(); // Open the chatbot ChatbotWidget.close(); // Close the chatbot ChatbotWidget.toggle(); // Toggle chatbot state ChatbotWidget.isOpen(); // Check if open ChatbotWidget.isReady(); // Check if ready // Dynamic configuration ChatbotWidget.updateConfig({ position: 'bottom-left', theme: 'light', toggleButtonText: 'Support' });

Advanced Features

  • Custom Styling: Override default styles with custom CSS
  • Event Handling: Listen to chatbot events and respond accordingly
  • Dynamic Configuration: Update widget settings at runtime
  • Programmatic Control: Open, close, or toggle the chatbot programmatically
  • Responsive Design: Automatically adapts to different screen sizes
  • Accessibility: Full keyboard navigation and screen reader support
