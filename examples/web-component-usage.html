Web Component Usage

This example shows how to use the chatbot widget as a web component.

You can place the <chatbot-widget> element anywhere in your HTML.

Basic Usage

<chatbot-widget api-url="https://your-api-domain.com" position="bottom-right" theme="light" primary-color="#3b82f6" button-color="#3b82f6" button-text-color="#ffffff" width="400px" height="700px" z-index="9999" show-welcome="false" welcome-message="Welcome to Our Support Chat"> </chatbot-widget>

Available Attributes

  • api-url - Your backend API URL
  • position - Widget position (bottom-right, bottom-left, top-right, top-left)
  • theme - Color theme (light, dark)
  • primary-color - Primary color for the widget
  • button-color - Chat button color
  • button-text-color - Chat button text color
  • width - Widget width (e.g., "400px")
  • height - Widget height (e.g., "700px")
  • z-index - CSS z-index value
  • show-welcome - Show welcome message (true/false)
  • welcome-message - Custom welcome message

JavaScript API

You can also control the widget programmatically:

// Get the widget element const widget = document.querySelector('chatbot-widget'); // Update configuration widget.updateConfig({ position: 'bottom-left', theme: 'dark', primaryColor: '#ff6b6b' }); // The widget will automatically update with the new configuration

Multiple Widgets

You can have multiple chatbot widgets on the same page:

<!-- Support widget --> <chatbot-widget api-url="https://support-api.com" position="bottom-right" primary-color="#3b82f6" welcome-message="Need help? Chat with support!"> </chatbot-widget> <!-- Sales widget --> <chatbot-widget api-url="https://sales-api.com" position="bottom-left" primary-color="#10b981" welcome-message="Interested in our products? Let's chat!"> </chatbot-widget>
