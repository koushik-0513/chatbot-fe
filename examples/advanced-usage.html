Widget Configuration

Interactive Demo

This example shows how to programmatically control the chatbot widget using JavaScript.

Use the buttons above to show, hide, update, or destroy the widget.

Modify the configuration settings and click "Update Config" to see changes in real-time.

API Methods:

  • window.ChatbotWidget.show() - Show the widget
  • window.ChatbotWidget.hide() - Hide the widget
  • window.ChatbotWidget.updateConfig(config) - Update widget configuration
  • window.ChatbotWidget.destroy() - Remove the widget completely
